# Muzily Infra

> 좋아하는 음악을 카페에서 틀고 싶다고? 🎧

### 개발 스택

- Language: Kotlin
- Framework: Spring Boot / Spring MVC / Spring Webflux
- Data: JPA / QueryDSL / MySQL / Redis
- Test: Junit5 / Spring Test / RestAssured
- Socket: STOMP
- Infra: k8s
- Operation: Grafana / Prometheus / Sentry
- CI/CD: Git / Github Actions / ArgoCD
